# Generar Iconos para PWA

Para generar los iconos necesarios para la PWA, puedes usar una herramienta online como:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

O crear iconos manualmente con estas especificaciones:

## Iconos Necesarios

1. **icon-192.png** - 192x192px
2. **icon-512.png** - 512x512px

Coloca estos archivos en la carpeta `public/`

## Diseño Sugerido

- Fondo: #6366f1 (indigo)
- Icono: Letra "N" o símbolo de reloj/calendario en blanco
- Formato: PNG con transparencia
- Bordes redondeados: Sí
